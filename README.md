# 项目 新闻小程序

## 项目概述

在这个项目中, 你将会从数据库中获取新闻，并以列表方式呈现。你完成的这个小程序，可以显示不同门类的热点新闻列表，也可以点击访问具体的新闻内容。你可以使用[我们准备好的 API](https://github.com/udacity/wmpnd-news/blob/master/news_api.md)，也可以使用真实的新闻数据。
